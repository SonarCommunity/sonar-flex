In Flex, the semicolon is optional as a statement separator, but omitting semicolons can be confusing.

Noncompliant Code Example

function fun() {
  return   // Noncompliant
       5   // Noncompliant
}
print(fun());  // prints "undefined", not "5"

Compliant Solution

function fun() {
  return 5;
}
print(fun());
